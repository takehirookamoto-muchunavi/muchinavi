## Step 1: 記事選定

### 手順
1. WordPress REST API (`GET /wp-json/wp/v2/posts?per_page=100`) で既存記事タイトル一覧を取得
2. スプレッドシートの記事候補一覧をブラウザで確認
3. 既存記事とタイトル・テーマが重複しないものをリストアップ
4. 優先順位判定（下記基準）で候補を絞り込み、ユーザーに提示

### 優先順位判定基準
1. 購買意向が高いターゲット層か（契約直前層・比較検討層を優先）
2. むちのちさんの「紹介ルート」差別化と相性が良いか
3. 既存記事との重複・カニバリがないか
4. 競合（SUUMO・HM公式ブログ）が手薄なテーマか

## Step 6: Agent M — CTA最適化

### 役割
Step 5完了後の記事に、MuchiNaviへの誘導文（CTA）を最適化して配置する。

### CTA設計方針
- **自然な流れ**: 記事の文脈に沿った誘導（押し売り感ゼロ）
- **読者メリット明確**: 「なぜLINE登録すると良いか」が伝わる
- **3パターン生成**: 異なるアプローチのCTAを3つ作り、最適版を選択

### CTA配置位置（3箇所）
1. **導入直後**: 読者が課題を認識したタイミング
2. **中盤の不安発生時**: 「失敗したくない」感情が動くポイント
3. **記事末尾**: 解決策として自然に提示

### 3パターン例

**パターンA: 問いかけ型**
> この記事を読んで「自分の場合はどうなんだろう？」と思った方は、LINEで無料相談してみませんか？24時間、あなたの状況に合わせてお答えします。

**パターンB: 損失回避型**
> 「あの時相談しておけばよかった…」と後悔しないために。まずはLINEで気になることを一つだけ聞いてみてください。

**パターンC: 具体的メリット型**
> あなたの年収や家族構成に合わせた具体的なシミュレーションが無料で受けられます。LINEに登録して、まずは気軽に聞いてみましょう。

### 出力
CTA3パターン + 推奨版 + 配置済み最終記事

## Step 7: Agent N — 否定レビュー（記事版）

### 役割
「読み逃げ読者」の視点で記事を徹底批判する。このステップを通さずに公開してはならない。

### 「読み逃げ読者」とは
- 記事を読んで「ふーん、参考になった」で閉じる読者
- LINE登録も面談予約もしない
- 「自分はまだ本格的に検討してない」と思っている

### チェック項目

#### 読者離脱ポイント
- [ ] 冒頭200文字で「この記事は自分に関係ある」と感じるか
- [ ] 専門用語の説明なしに使っていないか
- [ ] 段落が長すぎて読みにくくないか
- [ ] 「で、結局何をすればいいの？」に答えているか

#### CTA効果
- [ ] CTAが広告感丸出しで読者が引いていないか
- [ ] CTAの前に十分な価値提供があるか
- [ ] 「無料」「24時間」等のメリットが明確か
- [ ] 登録のハードルが高すぎないか

#### コンプライアンス
- [ ] 宅建業法違反の表現がないか
- [ ] 断定表現・誇大表現がないか
- [ ] 出典のない統計データがないか

#### コンテンツ品質
- [ ] SUUMO等の大手サイトと比べて独自の価値があるか
- [ ] 「むちのちに聞けばもっとわかる」と思わせるフックがあるか
- [ ] SNSでシェアしたくなる要素があるか

### 出力
agents/devil.md の標準フォーマットに準拠。「致命的問題なし」判定まで修正を繰り返す。

### 方法B: 手動入稿
REST APIが使えない場合、以下をユーザーに案内:
1. 最終記事テキストをクリップボードにコピー
2. WordPress管理画面で新規投稿
3. テキストを貼り付け
4. SWELL装飾を適用
5. アイキャッチ画像を設定
6. カテゴリ・タグを設定
7. メタディスクリプション・スラッグを入力
8. プレビュー確認後に公開

```
あなたは、共働きファミリーや子育て世帯に寄り添う不動産の専門家です。
note向けに、以下のテーマで【共感・ストーリー性・親しみやすさ】を重視した文章を書いてください。

【note記事の執筆ルール】
・文字数：1,200〜2,000文字程度
・一文を短くし、リズムよく読みやすい文章にしてください
・専門用語はなるべく避け、かみ砕いて説明してください

【構成テンプレート】
① 冒頭のつかみ（エモい導入／体験談／よくある悩みの共感）

【テーマ】
{テーマを記入}

【その他条件】
・最後に「ブログでも、家選びの考え方を詳しく書いています」と自然に誘導してください

```
